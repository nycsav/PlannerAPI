# signal2noise
**Daily AI intelligence for marketing strategists**

Agentic content intelligence platform that synthesizes signals from McKinsey, Gartner, BCG, Forrester, and 15+ sources into client-ready intelligence cards — delivered by 8am ET every weekday.

## What It Does
- Pulls from 15+ enterprise research sources daily
- Synthesizes into 3-4 signal cards with RWW scoring
- Categorizes: Competitive Signal, AI Signal, Media Signal, Brand Signal
- Publishes to signals.ensolabs.ai automatically
- Feeds the Enso Labs content engine (site, LinkedIn, newsletter)

## Stack
Firebase · React · Cloud Functions · Node.js
